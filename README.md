# 리뷰 
- 어려운 것 아니면 스스로 만드는 게 더 좋은 것 같다.
- 커스텀도 그렇고 SwiftUI에 딱 맞는 라이브러리가 아닌 이상 커스텀이 더 힘들다.
- 타이머를 쓰는 방법도 Combine이랑 그렇게 다르지 않으며 
- 가장 중요한 것은 네비게이션 이동이나, 로직이 끝난 후 반드시 타이머를 꺼줘야 하는 것이다.
